# 演讲评分系统

一个功能完整的演讲评分系统，支持多人协作、实时评分、数据管理等功能。

## 🚀 快速开始

### 本地使用

#### 方法1：直接打开（推荐）
1. 双击 `index.html` 文件
2. 在浏览器中打开即可使用

#### 方法2：使用本地服务器
1. 双击 `启动系统.bat` 文件
2. 系统会自动启动本地服务器并打开浏览器

### 部署到网上

## 📋 主要功能

### 1. 用户管理
- 添加/删除用户
- 批量导入用户（Excel）
- 角色分配（演讲者/评委）

### 2. 班级管理
- 创建/编辑/删除班级
- 固定班级（友好101、友好102、友好103）
- 班级切换

### 3. 评分系统
- 多维度评分（讲稿内容、语速、停顿强调、流利度、衣着形象、肢体语言）
- 去掉最高最低分求平均值
- 实时评分进度显示

### 4. 数据管理
- 三种存储模式：内存存储、本地存储、云端存储
- 导出Excel格式结果
- 历史记录查看

## 👥 用户角色

### 管理员
- 用户管理
- 班级管理
- 角色分配
- 开始评分
- 查看结果
- 导出数据

### 评委
- 查看演讲者列表
- 为演讲者评分
- 导出个人评分记录

### 演讲者
- 查看评分进度
- 查看最终得分

## 💾 存储模式

### 内存存储
- 数据临时保存在浏览器内存
- 刷新页面后数据丢失
- 适合临时测试

### 本地存储
- 数据持久化保存在浏览器
- 刷新页面后数据保留
- 适合单机使用

### 云端存储（默认）
- 数据保存在云端服务器
- 可以跨设备访问
- 适合多人协作

## 📖 使用流程

### 1. 管理员操作
```
1. 登录系统（管理员登录）
2. 选择或创建班级
3. 添加用户（手动或批量导入）
4. 分配角色（演讲者和评委）
5. 开始评分
6. 查看结果并导出
```

### 2. 评委操作
```
1. 登录系统（评委登录）
2. 输入会话ID
3. 为每位演讲者评分
4. 提交评分
5. 导出个人评分记录
```

### 3. 演讲者操作
```
1. 登录系统（演讲者登录）
2. 输入会话ID
3. 查看评分进度
4. 查看最终得分
```

## 🎯 固定班级

系统预设了三个固定班级，包含学生名单：

1. **友好101**（37人）
2. **友好102**（40人）
3. **友好103**（27人）

固定班级特点：
- 不能删除
- 不能编辑班级名称
- 不能清除用户
- 自动加载学生名单

## 📊 评分标准

| 评分项目 | 满分 | 说明 |
|---------|------|------|
| 讲稿内容 | 40分 | 内容质量、逻辑性、创新性 |
| 语速 | 10分 | 语速适中、节奏把控 |
| 停顿强调 | 10分 | 重点突出、停顿恰当 |
| 流利度 | 15分 | 表达流畅、无卡顿 |
| 衣着形象 | 10分 | 着装得体、仪表端庄 |
| 肢体语言 | 15分 | 手势自然、表情丰富 |
| **总分** | **100分** | |

## ⚠️ 注意事项

1. **固定班级保护**
   - 固定班级不能清除用户
   - 如需清除，请切换到其他班级

2. **数据备份**
   - 定期导出评分结果
   - 重要数据建议使用云端存储

3. **网络连接**
   - 云端存储需要网络连接
   - 本地存储和内存存储可离线使用

4. **浏览器兼容性**
   - 推荐使用Chrome或Edge浏览器
   - 确保浏览器版本较新

## 🆘 常见问题

### Q1: 如何批量导入用户？
**A**: 点击"批量添加"按钮，选择Excel文件（第一列为用户名）

### Q2: 评委看不到评分界面？
**A**: 确保管理员已点击"开始评分"，并检查会话ID是否正确

### Q3: 演讲者看到NaN？
**A**: 等待所有评委完成评分，系统会自动计算最终得分

### Q4: 如何切换存储模式？
**A**: 点击左上角数据管理面板，选择存储模式

### Q5: 固定班级如何清除用户？
**A**: 固定班级不能清除用户，请切换到其他班级或创建新班级
